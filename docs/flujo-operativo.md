# Flujo operativo

1. Llega una solicitud de personal.
2. Se crea un servicio con faena, fechas, turno y cargos requeridos.
3. El sistema filtra candidatos por cargo, turno, disponibilidad, acreditación y documentos.
4. Se genera la lista de llamados.
5. Se registran respuestas: confirmado, rechazado o sin respuesta.
6. Se crean asignaciones.
7. Se revisan acreditaciones y documentos pendientes.
8. Se determina si requiere pasaje.
9. Se hace seguimiento hasta que el servicio queda listo.
10. Al cierre del día se muestran tareas completadas y pendientes.
